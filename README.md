# Installation
1 Command
```
pkg update && pkg upgrade && pkg install git
```
2 Command
```
git clone https://github.com/rajbhx/cloudflared-termux 
```
3 Command
```
cd cloudflared-termux
```
4 Command
```
chmod +x Cloudflared-termux_@rajbhx.sh
```
5 Command
```
bash Cloudflared-termux_@rajbhx.sh
```
